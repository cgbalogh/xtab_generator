Wozu dient die Extension?
-------------------------

Kreuztabellengenerator zur Erstellung von Abfragen über beliebige Datensets und Tabellen.
Dieser Generator ist eine Weiterentwicklung der in webRelax5 4.5 eingesetzten Erweiterung, die sich in folgenden Punkten unterscheidet:

- Durchgängig generisch, also keine Abhängigkeit von relax5 8.7
- Verbesserte Konfigurationsmöglichkeiten
- Unterstützung mobiler Geräte

Einige EInstellungen sind allerdings nicht mehr möglich. So können aus Darstellungsgründen derzeit keine dreidimensionalen Ergebnislisten erzeugt werden (z-Achse).
